# 無限虚数学に基づく複合論理演算の提案
## 概要
　無限虚数学とは、無限を超越論的に取り扱う数学です。どういうことかというと、そのものとしての無限は我々の限定的な理解の形式を超えているので、我々が理解するところの無限とはつまり、「無限が無限として限定的に理解された無限である」という自己限定的前提に基づく無限、つまり無際限である、という前提に基づいて無限を取り扱う数学、それが無限虚数学であるということです。

　無限はいいけど、じゃあ虚数はなんなん？というと、通常の数学では、1 / ∞　は 0 として取り扱われる（らしい）のですが、無限虚数学においては、無限は無際限なので、実際のところ、実数として取り扱うことができます。したがって、無限虚数学においては、 1 / ∞ → 0　ただし、1 / ∞ ≠ 0 を満たすということになります。このとき、 1 / ∞ → 0　ただし、1 / ∞ ≠ 0 を満たす 1 / ∞ は、無際限に１から遠ざかり続けるが永遠に0に届かない存在として、0に極限まで近づきつつ、その極限において進行方向を虚数軸方向に90度転換し再び進み続けるという挙動を取ります（この挙動は、ブラックホールの特性として知られる事象の地平面と同じ性質を持つと言えます）。結果として、1 / ∞は、無限虚数学においては、虚数単位 i に等しいものとして取り扱われます。このような、我々が取り扱える無限とは超越論的無限としての無際限のみであるという前提に基づいて構築されたのが無限虚数学と、それに基づく複合論理演算です。

　（余談：僕は思うのですが・・・・もし無限を超越論的に取り扱わずに、超越的に取り扱おうとするのであれば・・・より正確に言えば、もし、我々の世界において無限が超越的に存在しているのであれば、それはすべてを無化するものでなくてはならないのではないか。もし、既存の数理論理体系が無限と真剣に向き合い、無限を制御しようと思うのであれば、無限を無限のまま野放しにしておけるはずがないと思うのです。一言でいえば、無限を（超越論的にではなく）超越的に、そのものとして取り扱おうとするならば、無限は数学を破壊、無化するものでなくてはならないということです。つまり、いかなる形であろうが、数式の中に一か所でも∞が出てきた時点で、少なくともその数式は無効にならなくてはならない。しかし現実問題、そうなっていない。それは現在の数学が、無限と無際限とを混乱の中で取り扱っているからです。僕は数学者でも数学研究者でもないのですが、今まで誰もこの問題を取り扱ってないのか？という疑問があり、絶対にそんなはずはないとも思うので、詳しい方は教えてください。基本的に無限虚数学は全く新しいものではなく、提示されている情報を全体としてつじつまが合うように整理していたら勝手に浮かび上がってきたものという認識です。なんとなくかっこいい名前をつけてみただけ。ただそれだけのことであり、そういう意味では逆に、非常に、持て余しています。）
　
　それで実際のところ、何がどう複合論理演算なのかというと、先ほども述べた通り、1 / ∞ = iという無限虚数学の定式は、存在を極限において取り扱うという点で相対論的極限論理であり、かつ、１＝１に基づくマクロスケールの古典論理と、１→０への動性を探るミクロスケールの量子論理を統合的に表現することができるため、三論理の統合としての複合論理演算であるということになります。この三論理は循環しており、それは、0と1を相対化する原点iの存在によって循環的に定義されます。
